# Reactific API

Auth + Stripe subscriptions + leaderboards for Number Hunt.

## Deploy to Render

1. Push this repo to GitHub
2. Render Dashboard → New → Blueprint
3. Connect repo → Render reads `render.yaml` and creates:
   - **reactific-api** (Node web service, $7/mo)
   - **reactific-db** (PostgreSQL, $7/mo)
4. Add environment variables in the service settings:
   - `STRIPE_SECRET_KEY` → from Stripe Dashboard
   - `STRIPE_WEBHOOK_SECRET` → from Stripe → Webhooks
   - `STRIPE_PRICE_ID` → from Stripe → Products → your monthly price
   - `CLIENT_URL` → `https://reactific.co`
   - `JWT_SECRET` → auto-generated by Render

## Set Up Database

After deploy, open the Render PostgreSQL shell and run:

psql $DATABASE_URL < schema.sql

Or paste the contents of `schema.sql` into the Render DB shell.

## Set Up Stripe Webhook

1. Stripe Dashboard → Developers → Webhooks
2. Add endpoint: `https://reactific-api.onrender.com/api/stripe/webhook`
3. Events to listen for:
   - `checkout.session.completed`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
4. Copy the webhook signing secret → set as `STRIPE_WEBHOOK_SECRET`

## API Endpoints

### Auth
- `POST /api/auth/register` — `{ email, username, password }`
- `POST /api/auth/login` — `{ email, password }`
- `GET /api/auth/me` — requires Bearer token

### Scores
- `POST /api/scores` — `{ court, speed, level, score, streak, tier }`

### Leaderboard
- `GET /api/leaderboard/daily?speed=slow&limit=50`
- `GET /api/leaderboard/weekly?speed=slow&limit=50`
- `GET /api/leaderboard/alltime?speed=slow&limit=50`
- `GET /api/leaderboard/myrank?speed=slow` — requires auth

### Stripe
- `POST /api/stripe/checkout` — creates checkout session, returns URL
- `POST /api/stripe/portal` — creates billing portal, returns URL
- `POST /api/stripe/webhook` — Stripe calls this automatically

### Health
- `GET /api/health` — DB connection check

## Frontend Integration

Replace the stub `cognitive-sync.js` on your static site with the real one from this repo. The game code doesn't change — `syncCognitiveScore()` is the same function signature.

Add `window.REACTIFIC_COURT = 'full'` in Full Court games and `window.REACTIFIC_COURT = 'half'` in Half Court games.
